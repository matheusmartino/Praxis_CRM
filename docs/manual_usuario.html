1. Introdução

O Praxis CRM é um sistema de gestão de relacionamento com clientes desenvolvido para equipes comerciais. Ele permite gerenciar leads, acompanhar oportunidades de vendas e monitorar a performance do time.

Principais funcionalidades:
  • Cadastro e gestão de clientes (leads)
  • Pipeline de vendas com etapas personalizadas
  • Registro de interações com clientes
  • Metas comerciais e acompanhamento de performance
  • Dashboards para gestão

2. Perfis de Usuário

O sistema possui três perfis de acesso, cada um com permissões específicas:

Perfil Descrição Principais Acessos
VENDEDOR Responsável pela prospecção e fechamento de vendas Seus próprios clientes, oportunidades e metas
GESTOR Acompanha a performance do time comercial Visualização de todos os dados (somente leitura)
ADMIN Administrador do sistema Acesso total, incluindo configurações

2.1 Comparativo de Permissões

3. Acesso ao Sistema

3.1 Login

Para acessar o sistema:

  1. Abra o navegador e acesse o endereço do sistema
  2. Na tela de login, informe seu usuário e senha
  3. Clique em Entrar
Importante: Caso esqueça sua senha, entre em contato com o administrador do sistema para redefinição.

3.2 Menu de Navegação

Após o login, você verá a barra de navegação no topo da tela com os seguintes itens:

Menu Descrição Disponível para
Clientes Lista e cadastro de clientes Todos
Oportunidades Lista e cadastro de oportunidades Todos
Interações Registro de contatos com clientes Todos
Minha Meta Acompanhamento da meta pessoal Vendedor
Gestão Dashboards e relatórios Gestor e Admin

4. Guia do Vendedor

4.1 Cadastrando um Cliente

  1. Acesse Clientes no menu
  2. Clique em Novo Cliente
  3. Preencha os dados obrigatórios:
    • Nome: Nome completo ou razão social
    • Tipo: B2B (empresa) ou B2C (pessoa física)
  4. Preencha os dados opcionais (telefone, e-mail, contato principal)
  5. Clique em Salvar
Dica: O cliente será criado com status Provisório. Isso permite cadastro rápido sem burocracia. O CNPJ/CPF será exigido apenas quando o admin for ativar o cliente.

4.2 Cadastrando uma Oportunidade

  1. Acesse Oportunidades no menu
  2. Clique em Nova Oportunidade
  3. Preencha:
    • Título: Nome descritivo da oportunidade
    • Cliente: Selecione o cliente relacionado
    • Valor estimado: Valor previsto da venda
    • Descrição: Detalhes adicionais (opcional)
  4. Clique em Salvar

4.3 Avançando no Pipeline

Cada oportunidade passa por etapas até o fechamento:

Prospecção → Qualificação → Proposta → Negociação → Fechamento

Para avançar uma oportunidade:

  1. Acesse a oportunidade clicando nela na lista
  2. Na tela de detalhes, clique em Avançar Etapa
  3. A oportunidade passará para a próxima etapa automaticamente
Atenção: Oportunidades marcadas como Perdida ou já em Fechamento não podem avançar.

4.4 Registrando Interações

Registre todos os contatos realizados com o cliente:

  1. Acesse Interações no menu
  2. Clique em Nova Interação
  3. Preencha:
    • Oportunidade: Selecione a oportunidade relacionada
    • Tipo: Ligação, E-mail, Reunião, WhatsApp ou Visita
    • Descrição: Resumo do que foi tratado
  4. Clique em Salvar

4.5 Acompanhando sua Meta

  1. Acesse Minha Meta no menu
  2. Visualize:
    • Meta: Valor definido para o mês
    • Realizado: Vendas fechadas no mês
    • Pipeline: Oportunidades em andamento
    • Status: Situação da meta (OK, Atenção ou Risco)

6. Guia do Administrador

O administrador possui acesso total ao sistema, incluindo o painel administrativo do Django.

6.1 Acessando o Django Admin

  1. Clique no seu nome no canto superior direito
  2. Selecione Admin
  3. Ou acesse diretamente: /admin/

6.2 Gerenciando Usuários

  1. No Admin, acesse Usuários
  2. Para criar um novo usuário:
    • Clique em Adicionar usuário
    • Defina usuário e senha
    • Na próxima tela, defina o Perfil (Vendedor, Gestor ou Admin)
    • Marque É da equipe se o usuário precisar acessar o Admin

6.3 Ativando Clientes

Clientes criados por vendedores ficam com status Provisório até serem aprovados:

  1. No Admin, acesse Clientes
  2. Localize o cliente provisório
  3. Verifique se o CNPJ/CPF está preenchido
  4. Selecione o cliente e use a ação Ativar clientes selecionados
Regra: O CNPJ/CPF é obrigatório para ativar um cliente. Caso não esteja preenchido, a ativação será bloqueada.

6.4 Cadastrando Metas

  1. No Admin, acesse Metas Comerciais
  2. Clique em Adicionar meta comercial
  3. Preencha:
    • Vendedor: Selecione o vendedor
    • Mês: Mês da meta
    • Ano: Ano da meta
    • Valor da Meta: Valor monetário
  4. Clique em Salvar
Regra: Cada vendedor pode ter apenas UMA meta por mês/ano. Se tentar criar uma duplicada, o sistema bloqueará.

7. Pipeline de Vendas

O pipeline é o caminho que uma oportunidade percorre desde o primeiro contato até o fechamento da venda.

7.1 Etapas do Pipeline

Prospecção

Primeiro contato com o potencial cliente. Ainda não há demanda validada.

Exemplos: Lead frio, indicação, primeiro contato via WhatsApp.

Qualificação

O vendedor validou que o lead tem potencial real de compra.

Exemplos: Confirmação de volume, identificação do decisor, pedido de cotação.

Proposta

Uma proposta comercial foi apresentada ao cliente.

Exemplos: Orçamento enviado, proposta formal por e-mail.

Negociação

Cliente demonstrou intenção de fechar, mas negocia condições.

Exemplos: Ajuste de preço, negociação de prazos.

Fechamento

Venda concluída com sucesso.

Exemplos: Pedido confirmado, contrato assinado.

Perdida

Venda não realizada.

Exemplos: Cliente desistiu, concorrente venceu.

8. Metas Comerciais

8.1 Como funcionam as Metas

As metas são definidas mensalmente pelo administrador para cada vendedor.

8.2 Cálculo do Realizado

Realizado = Soma dos valores das oportunidades com status Fechamento no mês/ano.

8.3 Cálculo do Pipeline

Pipeline = Soma dos valores das oportunidades em andamento (excluindo Fechamento e Perdida).

8.4 Status da Meta

Status Condição Significado
OK Pipeline ≥ 1,5× a meta Boa cobertura de pipeline
ATENÇÃO Pipeline ≥ meta Cobertura mínima
RISCO Pipeline < meta Necessita ação urgente
Por que 1,5×? Nem todas as oportunidades se convertem em venda. Ter um pipeline de pelo menos 1,5× a meta garante uma margem de segurança para atingir o objetivo.

9. Perguntas Frequentes

Por que não consigo ver os clientes de outros vendedores?

Por segurança, cada vendedor vê apenas seus próprios clientes e oportunidades. Gestores e admins podem ver todos os dados.

Por que não consigo ativar um cliente?

Apenas administradores podem ativar clientes. Além disso, o cliente precisa ter o CNPJ/CPF preenchido.

Como marco uma oportunidade como perdida?

Na tela de detalhes da oportunidade, clique em Marcar como Perdida.

Posso editar uma meta depois de criada?

Sim, o administrador pode editar metas no Django Admin.

O que acontece se eu tentar criar uma meta duplicada?

O sistema não permite duas metas para o mesmo vendedor no mesmo mês/ano.

Como sou notificado sobre minha meta?

Acesse a tela Minha Meta regularmente para acompanhar seu progresso. O status (OK, Atenção, Risco) indica sua situação.
